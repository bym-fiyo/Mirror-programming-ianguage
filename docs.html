📖 Введение

Mirror Language — это современный язык программирования с синтаксисом на основе отступов, созданный для того, чтобы код был читаемым как обычный текст.

Основные принципы

  • Читаемость: Код должен читаться как псевдокод
  • Простота: Минимум синтаксического шума
  • Отступы: 4 пробела для вложенности (никаких фигурных скобок!)
  • Дружелюбность: Понятные сообщения об ошибках
📌 Примечание: Mirror использует отступы для определения блоков кода, подобно Python. Фигурные скобки {} используются только для словарей.

Способы установки

📝 Синтаксис

Отступы

Для определения блоков кода используются отступы:

  • 4 пробела = один уровень вложенности
  • Табуляция приравнивается к 4 пробелам
  • Несогласованные отступы вызывают ошибку
Правильно:
func greet(name):
    if name:
        echo("Hello, " + name)
    else:
        echo("Hello!")

Комментарии

Двоеточие

Двоеточие : используется после заголовков конструкций:

  • После параметров функции
  • После условия if/while
  • После ключевого слова try/catch/finally

🪞 Зеркальные функции (mirrorfunc)

💡 Новинка! Mirror — первый язык программирования с поддержкой зеркальных функций, выполняющих код в обратном порядке.

Что такое mirrorfunc?

Зеркальные функции выполняют операторы своего тела в обратном хронологическом порядке — от последнего к первому. Это уникальная особенность языка Mirror.

Синтаксис

mirrorfunc function_name(param1, param2):
    # Оператор 1
    # Оператор 2
    # Оператор 3
    ret value

Пример использования

Сравнение обычной и зеркальной функции
# Обычная функция
func normal():
    echo("first")
    echo("second")
    echo("third")

# Зеркальная функция
mirrorfunc mirror():
    echo("first")
    echo("second")
    echo("third")

normal()
# Вывод:
# first
# second
# third

mirror()
# Вывод:
# third
# second
# first

Когда использовать?

  • 🔄 Обратные операции — отмена действий, rollback транзакций
  • 🧹 Очистка ресурсов — закрытие файлов, освобождение памяти
  • 🧪 Тестирование — проверка инвариантов при разрушении состояния
  • 🎨 Креативное программирование — необычные алгоритмы и эффекты

Практический пример

Управление ресурсами
# Зеркальная функция для очистки
mirrorfunc cleanup():
    echo("Closing file...")
    echo("Releasing memory...")
    echo("Disconnecting...")

cleanup()
# Вывод:
# Disconnecting...
# Releasing memory...
# Closing file...

Возврат значения

mirrorfunc reverse_add(a, b):
    echo("Computing...")
    ret a + b

var result = reverse_add(2, 3)
# Вывод: Computing...
# result = 5
💡 Совет: Зеркальные функции используют ключевое слово mirrorfunc (слитно). Отступы работают так же, как в обычных функциях — 4 пробела на уровень вложенности.

🔧 Макросы (macro)

💡 Новинка! Макросы позволяют создавать код, который генерирует другой код на этапе выполнения.

Что такое макросы?

Макросы — это механизм метапрограммирования, который позволяет писать код, выполняющийся до основной программы. В отличие от функций, макросы разворачиваются в момент вызова и могут генерировать код динамически.

Синтаксис

macro name(param1, param2):
    # Тело макроса
    # Использует параметры как переменные
    ret result

Вызов макросов

Макросы вызываются с восклицательным знаком ! перед скобками:

name!(arg1, arg2)

Пример использования

Макрос отладки
# Макрос для отладки переменных
macro debug(var_name):
    echo("DEBUG: " + var_name + " = ")
    ret var_name

var x = 42
debug!(x)
# Вывод:
# DEBUG: x = 
# 42

Макрос unless

Обратный if
macro unless(condition, code):
    if not condition:
        ret code
    ret nil

var age = 15
unless!(age >= 18, echo("Minor"))
# Вывод: Minor

Макрос repeat

Повторение кода
macro repeat(n, code):
    for i in range(n):
        code
    ret nil

repeat!(3, echo("Hello!"))
# Вывод: Hello! (3 раза)

Когда использовать макросы?

  • 🐛 Отладка — автоматическое логирование с именами переменных
  • 📝 DSL — создание предметно-ориентированных языков
  • 🔄 Генерация кода — автоматическое создание шаблонного кода
  • ⚡ Оптимизация — вычисления на этапе компиляции

Макросы vs Функции

ХарактеристикаФункцияМакрос
Когда выполняетсяRuntimeДо выполнения
Что принимаетЗначенияКод (как значения)
Вызовname()name!()
Возвратret valueret value
⚠️ Важно: Аргументы макроса вычисляются до передачи в макрос. Это означает, что код аргумента выполняется один раз перед вызовом макроса.
💡 Совет: Используйте макросы для устранения дублирования кода и создания выразительных DSL. Но не злоупотребляйте — слишком много макросов могут усложнить понимание кода.

Простой калькулятор

func calc(a, op, b):
    switch op:
        case "+":
            ret a + b
        case "-":
            ret a - b
        case "*":
            ret a * b
        case "/":
            ret a / b
        default:
            ret nil

echo(calc(10, "+", 5))  # 15
