## Для особо одарённых - инструкция по установке:
1. Скачайте с сайта https://mirror-programming-language.netlify.app файл install.zip (это архив с программой установки)
2. Распакуйте архив в удобную вам папку
3. Вам нужен файл install-gui.py. ОН ОБЯЗАТЕЛЬНО ДОЛЖЕН ЛЕЖАТЬ В ПАПКЕ С ИСХОДНИКАМИ
4. Запустите файл install-gui.py
5. Если при запуске возникла ошибка, следуйте инструкциям на сайте https://mirror-programming-language.netlify.app (установите PyQt6)
6. Готово! Вопросы? Пишите мне на почту, разберёмся!
